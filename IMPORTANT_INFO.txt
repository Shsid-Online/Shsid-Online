SHSID Social Platform - Important Implementation Notes
Last updated: 2026-05-07

1) Current signup flow (frontend)
- Step 1: Email entry
- Step 2: Verification step (Firebase-based)
  - "Send verification email" sends Firebase verification email.
  - User enters verification code in app.
  - "Verify code" applies code via Firebase Auth applyActionCode().
- Step 3: Password with double confirmation
- Step 4: Name + class info
  - Year must be 1-12
  - Class must be 1-13
- Step 5: Video upload (verification video filename submitted)
- Step 6: Waiting page (wait for admin approval)

2) Login behavior
- Login uses backend API endpoint /api/auth/login.
- Firebase login check was removed from login path because it broke normal backend login flow.
